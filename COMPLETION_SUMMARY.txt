## ⚡ Performance Notes

### Training Time
- **Baseline** (5 models, 5-fold CV): 5-10 minutes
- **With all 8 models**: 10-15 minutes
- **With hyperparameter tuning**: 30+ minutes

### Memory Usage
- **Baseline**: ~2-3 GB RAM
- **With GPU**: Uses VRAM instead
- **Reducible**: Use fewer models, reduce cv_splits, adjust subsample

### Optimization Tips
1. Use fewer models during development
2. Reduce cross-validation folds (5 → 3)
3. Reduce quantile subsampling
4. Enable GPU if available
5. Use result caching
